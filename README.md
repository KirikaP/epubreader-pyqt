## Quick Start

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python main.py
```

## Shortcuts

- Ctrl+O: Open file
- Ctrl+T: Toggle table of contents
- Ctrl+M: Toggle reading mode
